## その他の顔をなくすボタンについて
顔をなくすボタンを作られた場合には是非ともリンクさせていただきたいです。`src/_data/othernofacebuttons.json`中の`others`配列をよしなに編集してPRを送っていただければと思います。例えばこのようなものとなります。

```json
{
    "name": "顔をなくすボタン",
    "url": "https://noface.osumiakari.jp",
    "author": "Osumi Akari"
}
```
